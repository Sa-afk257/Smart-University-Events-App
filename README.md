# CampusConnect – Smart University Events App 🚀

## Overview

CampusConnect is an Android application developed for the ENCS5150 Android Course Project.

The application helps university students discover workshops, seminars, competitions, conferences, and student activities in a modern and user-friendly environment.

Users can browse events, make reservations, save favorites, manage profiles, and receive updated event information through REST API integration.
